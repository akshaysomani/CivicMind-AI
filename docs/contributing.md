# Contributing to CivicMind AI

Thank you for your interest in contributing to CivicMind AI! We welcome contributions from developers, designers, writers, and community builders.

## Code of Conduct
Please review and adhere to our [Code of Conduct](code_of_conduct.md) to keep this community safe and welcoming.

## Getting Started
1. Fork the repository on GitHub.
2. Clone your fork locally:
   ```bash
   git clone https://github.com/your-username/CivicMind-AI.git
   ```
3. Initialize the development environment by following the [Installation Guide](installation.md).
4. Create a new branch for your edits:
   ```bash
   git checkout -b feature/amazing-new-feature
   ```

## Development Guidelines
- **TypeScript & React:** Ensure zero compilation warnings or type issues. Run `npm run build` before pushing.
- **Python Backend:** Write clean code conforming to PEP 8 standards. Run `python -m pytest` to verify all tests pass.
- **Commit Messages:** Use structured commits (e.g. `feat(analytics): add ward analytics page`).

## Pull Request Process
1. Complete the checklist in our [Pull Request Template](../.github/pull_request_template.md).
2. Ensure test coverage remains high.
3. Submit the PR to the `main` branch. A maintainer will review your changes shortly!

# Changelog

All notable changes to the CivicMind AI project will be documented in this file.

## [1.0.0] - 2026-06-25
### Added
- **Multi-Agent AI Orchestration:** Integrated Gemini semantic router with specialized GIS, health, schemes, and general assistance agents.
- **GIS Map Layering:** Added interactive Leaflet map rendering geo-tagged citizen concerns, custom ward boundaries, and heatmaps.
- **Enterprise Testing (QA):** Created a comprehensive test suite of 108 unit/integration tests running with isolated sqlite databases.
- **Demo / Presentation Mode:** Launched an interactive Hackathon Guided Tour with persona auto-authentication, active state indicators, and floating banners.
- **Reporting & Workflow Builder:** Custom Excel/CSV/PDF exports for government triages, and a drag-and-drop workflow configuration engine.
- **Accessibility & Themes:** Implemented full Dark/Light color contrast transitions conforming to WCAG AA parameters.
